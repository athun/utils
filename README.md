# Reference

## print_table

#### *Description*

Prints a table from a given array of arrays of strings. Programmer can customize the look of the table.

#### *Sample*
~~~~
[alexander@mycoolpc utils]$ ./print_table.py
╔═══════════╦═══════════╦═══════════╦═══════════╗
║ o         ║ o         ║ x         ║           ║
╠═══════════╬═══════════╬═══════════╬═══════════╣
║           ║ x         ║           ║           ║
╠═══════════╬═══════════╬═══════════╬═══════════╣
║ 0         ║ x         ║           ║ alexander ║
╚═══════════╩═══════════╩═══════════╩═══════════╝
~~~~
